Species

The figure below is a bar plot of species of penguins.

ggplot(
  penguins,
  aes(
    x = bill_length_mm, y = bill_depth_mm,
    color = species, shape = species
  )
) +
  geom_point() +
  theme_minimal() +
  scale_color_colorblind() +
  labs(x = "Bill length (mm)", y = "Bill depth (mm)")
